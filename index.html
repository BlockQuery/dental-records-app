GitHub 数据同步配置

配置GitHub仓库用于云端存储病例数据，实现多设备数据同步。请先在GitHub创建一个私有仓库并生成Personal Access Token。

💡 本地运行：通过「启动服务器.bat」启动时，项目根目录的 github数据库私钥.txt 会自动填入下方 Token，无需手动粘贴（注意：直接双击 index.html 的 file:// 协议会被浏览器拦截 GitHub 请求）。
🌐 公网部署：服务端不含私钥文件，必须在此处手动填写 Token 才能同步数据，请勿把私钥文件上传到公开托管。填写后点「保存配置」即可。

Token需要 repo 权限。数据存储在您的私有仓库中，完全由您掌控。
